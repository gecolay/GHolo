## Local development

### Local project

Clone the repository:
```bash
git clone https://github.com/Gecolay/GHolo.git
```

### Build

Run the `gradle build` command.

The final `GHolo-x.x-x.jar` file will be in the [`build/libs`](./build/libs) folder.
